    –°–æ–≤–º–µ—Å—Ç–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å¶

    –°–æ–≤–º–µ—Å—Ç–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –¥–≤—É—Ö —Å–æ–±—ã—Ç–∏–π \(A\) –∏ \(B\) –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è –∫–∞–∫ \(P(A \cap B)\) –∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ –æ–±–∞ —Å–æ–±—ã—Ç–∏—è –ø—Ä–æ–∏–∑–æ–π–¥—É—Ç –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ.

    –£—Å–ª–æ–≤–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å¶

    –£—Å–ª–æ–≤–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è \(A\) –ø—Ä–∏ —É—Å–ª–æ–≤–∏–∏, —á—Ç–æ –ø—Ä–æ–∏–∑–æ—à–ª–æ —Å–æ–±—ã—Ç–∏–µ \(B\), –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è –∫–∞–∫ \(P(A|B)\) –∏ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è –∫–∞–∫: \(P(A|B) = \dfrac{P(A \cap B)}{P(B)}, \quad \text{–≥–¥–µ} \quad P(B) > 0\)

    –§–æ—Ä–º—É–ª–∞ –ë–∞–π–µ—Å–∞¶

    –§–æ—Ä–º—É–ª–∞ –ë–∞–π–µ—Å–∞ –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–µ—Ä–µ—Å—á–∏—Ç—ã–≤–∞—Ç—å —É—Å–ª–æ–≤–Ω—ã–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ —Å–æ–±—ã—Ç–∏–π –Ω–∞ –æ—Å–Ω–æ–≤–µ –Ω–æ–≤–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏. –° —É—á–µ—Ç–æ–º —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ \(P(A \cap B) = P(A|B) P(B) = P(B|A) P(A)\) —Ñ–æ—Ä–º—É–ª–∞ –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è —Ç–∞–∫:

    \(P(A|B) = \dfrac{P(B|A) \cdot P(A)}{P(B)}, \quad \text{–≥–¥–µ} \quad P(B) > 0\)

    –ü—Ä–∏–º–µ—Ä –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è —Ñ–æ—Ä–º—É–ª—ã –ë–∞–π–µ—Å–∞¶

    –†–∞—Å—Å–º–æ—Ç—Ä–∏–º –ø—Ä–∏–º–µ—Ä. –ü—É—Å—Ç—å –∏–º–µ–µ—Ç—Å—è —Ç–µ—Å—Ç –Ω–∞ –±–æ–ª–µ–∑–Ω—å, –∫–æ—Ç–æ—Ä—ã–π –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–º –∏–ª–∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º. –ü—É—Å—Ç—å: - \(A\) ‚Äî —Å–æ–±—ã—Ç–∏–µ, —á—Ç–æ —á–µ–ª–æ–≤–µ–∫ –±–æ–ª–µ–Ω. - \(B\) ‚Äî —Å–æ–±—ã—Ç–∏–µ, —á—Ç–æ —Ç–µ—Å—Ç –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–π.

    –î–∞–Ω—ã —Å–ª–µ–¥—É—é—â–∏–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏: - –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –±–æ–ª–µ–∑–Ω–∏ –≤ –ø–æ–ø—É–ª—è—Ü–∏–∏: \(P(A) = 0.01\). - –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–≥–æ —Ç–µ—Å—Ç–∞ —É –±–æ–ª—å–Ω–æ–≥–æ —á–µ–ª–æ–≤–µ–∫–∞ (—á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Ç–µ—Å—Ç–∞): \(P(B|A) = 0.99\). - –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–≥–æ —Ç–µ—Å—Ç–∞ —É –∑–¥–æ—Ä–æ–≤–æ–≥–æ —á–µ–ª–æ–≤–µ–∫–∞ (–ª–æ–∂–Ω–æ–ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç): \(P(B|\neg A) = 0.05\).

    –ù–µ–æ–±—Ö–æ–¥–∏–º–æ –Ω–∞–π—Ç–∏ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ —á–µ–ª–æ–≤–µ–∫ –±–æ–ª–µ–Ω –ø—Ä–∏ —É—Å–ª–æ–≤–∏–∏, —á—Ç–æ —Ç–µ—Å—Ç –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–π, —Ç–æ –µ—Å—Ç—å \(P(A|B)\).

    –î–ª—è —ç—Ç–æ–≥–æ —Å–Ω–∞—á–∞–ª–∞ –Ω–∞–π–¥—ë–º \(P(B)\) ‚Äî –ø–æ–ª–Ω—É—é –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–≥–æ —Ç–µ—Å—Ç–∞: \(P(B) = P(B|A) \cdot P(A) + P(B|\neg A) \cdot P(\neg A)\) \(P(B) = 0.99 \cdot 0.01 + 0.05 \cdot 0.99\) \(P(B) = 0.0099 + 0.0495 = 0.0594\)

    –¢–µ–ø–µ—Ä—å –∏—Å–ø–æ–ª—å–∑—É–µ–º —Ñ–æ—Ä–º—É–ª—É –ë–∞–π–µ—Å–∞:

    \(P(A|B) = \dfrac{P(B|A) \cdot P(A)}{P(B)}\)

    \(P(A|B) = \dfrac{0.99 \cdot 0.01}{0.0594}\)

    \(P(A|B) = \dfrac{0.0099}{0.0594} \approx 0.1667\)

    –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ —á–µ–ª–æ–≤–µ–∫ –±–æ–ª–µ–Ω –ø—Ä–∏ —É—Å–ª–æ–≤–∏–∏, —á—Ç–æ —Ç–µ—Å—Ç –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–π, —Å–æ—Å—Ç–∞–≤–ª—è–µ—Ç –ø—Ä–∏–±–ª–∏–∑–∏—Ç–µ–ª—å–Ω–æ 0.167, –∏–ª–∏ 16.67%.

    –≠—Ç–æ—Ç –ø—Ä–∏–º–µ—Ä –∏–ª–ª—é—Å—Ç—Ä–∏—Ä—É–µ—Ç, –∫–∞–∫ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ —Ñ–æ—Ä–º—É–ª—ã –ë–∞–π–µ—Å–∞ –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ–±–Ω–æ–≤–∏—Ç—å –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è –Ω–∞ –æ—Å–Ω–æ–≤–µ –Ω–æ–≤–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏, —á—Ç–æ —è–≤–ª—è–µ—Ç—Å—è –≤–∞–∂–Ω—ã–º –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–æ–º –≤ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –æ–±–ª–∞—Å—Ç—è—Ö –Ω–∞—É–∫–∏ –∏ —Ç–µ—Ö–Ω–∏–∫–∏.

    from abc import ABC, abstractmethod

# –°–æ–∑–¥–∞–µ–º –∞–±—Å—Ç—Ä–∞–∫—Ç–Ω—ã–π –∫–ª–∞—Å—Å –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å —Ñ–æ—Ä–º—É–ª–æ–π –ë–∞–π–µ—Å–∞
class BayesFormula(ABC):
    def __init__(self, prob_a, prob_b_given_a, prob_b_given_not_a):
        self.prob_a = prob_a  # –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è A
        self.prob_b_given_a = prob_b_given_a  # –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è B –ø—Ä–∏ —É—Å–ª–æ–≤–∏–∏ A
        self.prob_b_given_not_a = prob_b_given_not_a  # –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è B –ø—Ä–∏ —É—Å–ª–æ–≤–∏–∏ –Ω–µ A

    @abstractmethod
    def calculate_posterior(self):
        pass

# –ö–æ–Ω–∫—Ä–µ—Ç–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è –∫–ª–∞—Å—Å–∞ –¥–ª—è —Ñ–æ—Ä–º—É–ª—ã –ë–∞–π–µ—Å–∞
class BayesCalculator(BayesFormula):
    def calculate_posterior(self):
        # –í—ã—á–∏—Å–ª—è–µ–º –ø–æ–ª–Ω—É—é –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è B
        total_prob_b = self.prob_b_given_a * self.prob_a + self.prob_b_given_not_a * (1 - self.prob_a)
        # –í—ã—á–∏—Å–ª—è–µ–º –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è A –ø—Ä–∏ —É—Å–ª–æ–≤–∏–∏ B –ø–æ —Ñ–æ—Ä–º—É–ª–µ –ë–∞–π–µ—Å–∞
        posterior = (self.prob_b_given_a * self.prob_a) / total_prob_b
        return posterior

# –ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞ –¥–ª—è –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –±–æ–ª–µ–∑–Ω–∏ –ø—Ä–∏ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–º —Ç–µ—Å—Ç–µ
if __name__ == "__main__":
    # –î–∞–Ω–æ:
    prob_disease = 0.01  # –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –±–æ–ª–µ–∑–Ω–∏ –≤ –ø–æ–ø—É–ª—è—Ü–∏–∏
    prob_positive_given_disease = 0.99  # –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–≥–æ —Ç–µ—Å—Ç–∞ —É –±–æ–ª—å–Ω–æ–≥–æ —á–µ–ª–æ–≤–µ–∫–∞

    # –°–æ–∑–¥–∞–µ–º –æ–±—ä–µ–∫—Ç –∫–ª–∞—Å—Å–∞ BayesCalculator —Å –∑–∞–¥–∞–Ω–Ω—ã–º–∏ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—è–º–∏
    bayes = BayesCalculator(prob_disease, prob_positive_given_disease, prob_positive_given_no_disease)

    # –í—ã—á–∏—Å–ª—è–µ–º –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –±–æ–ª–µ–∑–Ω–∏ –ø—Ä–∏ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–º —Ç–µ—Å—Ç–µ
    posterior_probability = bayes.calculate_posterior()

    # –í—ã–≤–æ–¥–∏–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç
    print("–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ —á–µ–ª–æ–≤–µ–∫ –±–æ–ª–µ–Ω –ø—Ä–∏ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–º —Ç–µ—Å—Ç–µ:", posterior_probability)
